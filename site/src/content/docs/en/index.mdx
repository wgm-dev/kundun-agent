## Install in seconds

```bash
npm install -g kundun-agent
kundun init
kundun scan
```

Then connect it to your coding agent:

```bash
claude mcp add kundun-agent -- kundun --project-root /path/to/your/project mcp
```

## Why it exists

Coding agents waste context re-reading files they have already seen and forget
decisions between sessions. Kundun-Agent gives them a fast, local index plus a
persistent memory and task store — so they retrieve exactly what they need and
build on what was decided before.
