## Instale em segundos

```bash
npm install -g kundun-agent
kundun init
kundun scan
```

Depois conecte ao seu agente de programação:

```bash
claude mcp add kundun-agent -- kundun --project-root /caminho/do/seu/projeto mcp
```

## Por que existe

Agentes de programação gastam contexto relendo arquivos que já viram e esquecem
decisões entre sessões. O Kundun-Agent dá a eles um índice local rápido mais uma
memória e tarefas persistentes — então recuperam exatamente o que precisam e
constroem sobre o que já foi decidido.
